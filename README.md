# Ni Tinnu 🍕🚴

**Pedal-Powered Sourdough Pizza**

A community-driven pizzeria in Bangalore bringing together cycling culture, wood-fired pizza, and slow evenings.

🌐 **[nitinnu.in](https://nitinnu.in)**

## About

Ni Tinnu started as a simple idea between friends — good pizza, cycling rides, and a place to belong. We pop up in different corners of Bangalore, serve fresh wood-fired pizzas, and create memories that last longer than a meal.

*Come for the pizza, stay for the people.*
